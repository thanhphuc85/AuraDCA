How it works
Claude recommends. Code decides.

The LLM is never the final authority on how much money moves.

1
Claude decidesGiven balance, budget, day count & history, Claude returns a proposed amount + reasoning via a forced, schema-validated tool call.
2
Guardrails clampA pure, unit-tested function re-derives the cap from hard limits (max/day, reserve, dust, budget). The sole spending authority.
3
Swap & recordCircle Swap Kit executes USDC → cirBTC on Arc Testnet; the run is appended to a public audit trail committed back to the repo.
