How it works

Claude recommends; code decides. The LLM is never the final authority on how much money moves.

1 · Claude decidesGiven balance, budget, day count & history, Claude returns a proposed amount + reasoning via a forced, schema-validated tool call.
2 · Guardrails clampA pure, unit-tested function re-derives the cap from hard limits (max/day, reserve, dust, budget). This is the sole spending authority.
3 · Swap & recordCircle Swap Kit executes USDC → cirBTC on Arc Testnet; the run is appended to a public audit trail committed back to the repo.

The autonomous trading runs server-side (a GitHub Actions cron with a Circle Developer-Controlled Wallet), so no API keys are ever exposed to the browser. This page is a read-only dashboard: connecting your wallet only reads your Arc balance.
